Hosting Options

Aerial offers three hosting options - the Aerial Bots, which are free and available to everyone, but Self Hosting is also supported. If you boost the Discord Server, you will receive a bot on Cloud Hosting, online 24/7.

Aerial Bots

Aerial has 256 disposable bots, available for you within seconds in our Discord Server. Just type start in #create-a-bot to get started!

Self Hosting

Aerial also has a self hosted client, which is available for download on our GitHub Page. Just download the client, unzip it, edit the configuration file, install packages, and run it!
